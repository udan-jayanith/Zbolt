# API-Client
This is an API-Client like Postman and Insomnia. Other API clients use Javascript which makes them slow. This repo mainly aims to make an faster API client.

## This application aims to implement 
* HTTP
* GraphQL
* GRPC

This application uses experimental Guigui module for gui development and it lacks many UI component becuase of that I'm mainly developing UI part of the application without functionalities. After that i have planned to make HTTP and GraphQL functional.
